# WeazlFeed

Sovereign, local-first RSS/Atom TUI and podcast player.

## Install

```sh
./scripts/install.sh
```

## Run

```sh
weazlfeed
```

Config lives at `~/.config/weazlfeed/config.json` unless `WEAZLFEED_CONFIG` is set.
Data lives at `~/.local/share/weazlfeed/weazlfeed.sqlite3` unless `WEAZLFEED_DATA` is set.
